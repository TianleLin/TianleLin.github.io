Essay-one

发表于 2020-03-09

​ 今天一觉醒来发现屋外已是惊风雨，便想随便写点什么。

​ 每天折腾我那 Ubuntu 依然乐此不疲，代码倒是没怎么写，闲书也不看了，所以，这时我应该略略停下来，写点什么，深呼吸，看一看外面的绿叶，毕竟，春天终归是来了。

​ 功课当然是不容易，只是，远没有那么大的心理负担，日子还是一天天过，每天也都能学到些什么，尽管没有其他人学得那么好，但所幸没有丧失热情，没有失了好奇，我可以时不时做一些感兴趣的，所谓的课外活动。明明是源于好奇并想填补这些无知，可却获得了更多的无知与好奇，真是矛盾呢。

​ 之前想买的东西每天都想不起来买，明明都已经收藏放进了购物车，可宅在家里几乎没有什么对这些的兴趣了。还好，没有放弃说话，坚持打电话，甚至是自言自语，以免出了家门，便无话可说，支支吾吾，像极了书呆子。

​ 拇指琴的练习也只是停留在了二三曲子，每天也只是上上课写写作业，定时吃饭睡觉游戏，定时聊天，不时想念。

​ 过几天把 Ubuntu 折腾笔记2再记一记，当然也不限于 Linux 的范围，只是把我的学习路径简单的回忆一下，以免遗忘，总还是要留下点什么。

​ 当然，功课的反思还是要做的，不能拣了芝麻丢了西瓜，只是可以不用写在这里，仅给自己看看就好。

​ 最近愈发地显得暴躁，或是无聊，不知道我要做些什么，因此，谨以此文提点自己一二，使我不必再想三四。
