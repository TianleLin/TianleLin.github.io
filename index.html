The Third StarStudio Weekly Newspaper

发表于 2019-11-16

本周任务：一个LOGO

前期准备：

  • 在网上寻找相关的logo寻找灵感，关键词：star, stars, stars logo.

  • 选取各logo中主要的元素进行重构

  • 产生最终构思（形状，色调，字体）

中期制作：

构思完logo的大致样式了后，制作的步骤就会有针对性。因此先查找有关的ps、ai教程，熟练操作。

但是！

发现并不能足够高效的用ps和ai。(:з」∠)

在查找相关教程后，宣布放弃。

改用 Autodesk SketchBook , 直接画出来。

因此，在一通操作后：

  1. 画出logo的背景形状，填充颜色。
  2. 画出logo内部的主体内容（星星），填充颜色（此处我错误的滥用莫兰迪色，产生土俗效果）
  3. 加上名字（StarStudio）
  4. 添加光晕

成品（说实话改进空间巨大，以后再重制），耗时 2h。

后期反思：

虽然在deadline之前成功出稿子，但成品仍显得很不成熟，模仿痕迹还是很重，色彩的选取不够和谐。

硬伤是并没有用上ps和ai，即所学的知识并未派上实际用场，未能转化为生产力。

不过对笔的使用倒是更熟练了，大部分线条都是一遍过，不用多次修改或是重画。手更稳了（笑）。

然后萌发了想买各种带手写笔的电子产品的想法。完了以后要吃土了（逃）。

其他

  • 换了新的梯子，稳定快速并且贵(:з」∠)
  • 启用 telegram 并开始规划一个完善的群组，想制作一个bot，参考： 如何制作一个聊天机器人
  • 推荐教程的补番 1、一个标志的演变 2、标志设计中常用的网格

加入星辰的第一周学习

重点是Photoshop的绘画应用

Tutorial Video： How to EASILY Draw 2D landscapes in Photoshop

预期效果

其中使用到多种有用且好记的快捷键，以及一种选色技巧。

  • Window> Find extension exchange> Adobe color theme。
    • 其中有选色rear。
    • 只需选择一个颜色即可创造不同灰度与色调的颜色。
  • 一切行动前先新建图层（Ctrl + right click）
  • 使用套索工具（L）勾出轮廓，”Then Fill a layer”(Alt + backspace)[foreground] or (Ctrl + backspace)[background] 并选择solid color。
  • 在 Color picker 中只选中S 或 B 调整 亮度与更深/浅的颜色。
  • 调出curve（Ctrl + M）
  • “Do some shading here”遮罩图层：新建图层与上> Alt + click。
  • 制作树笔刷：Filter> Render> Tree（然后添加到笔刷就完了）可调整扰动和大小。
