Ubuntu-Toss-Record

发表于 2020-02-26

写在前面

​ 因为选修课的缘故，就要用到Ubuntu 18.04 LTS来写东西。老师给的要求是VMware+Ubuntu18+OpenCV。但目前的surface用虚拟机还是太吃力，不仅是运行还是存储上都不宽裕，因此用了别的奇技淫巧。

初次尝试 live USB

​ 没错，曲线救国x1，用Ubuntu的live USB模式将Ubuntu寄存在U盘中即插即用。

​ 但是，坑实在是太多了。此文将慢慢地回忆起我遇到的一个个莫名其妙的未解之谜，并自我推断问题原因。谨以此文告诫未来的自己，不要重复做傻事。

食用方法

​ 使用 Universal-USB-Installer-1.9.9.0 配合 ubuntu-18.04.4-desktop-amd64.iso 制作启动U盘。其中，需要设置Persistent storage。让U盘变成便捷Ubuntu系统 简书的这篇文章就是入门坑了。分别给了两篇文章参考，但明明用了两种方法。

​ 而我，选了日后埋下问题的那一种。HOW-TOHow to Create a Persistent Linux Installation on a Bootable Live USB or SD Card 步骤非常简单，做完即插即用。（此时我有两个U盘，并分别测试了速度，才发现我一直在用的U盘只是个2.0的）

开始折腾

​ 自己的想法是面对一个新系统，需要解决网络、输入的问题。

  1. v2ray（科学上网）。

    ​ 这时看到了油管上 Ubuntu系统如何使用v2ray客户端 一通操作用另一个电脑下下来必要的文件用U盘导入（为什么不直接下呢？因为没有科学上网下不下来，就形成了一个出不去的闭环）。可怜的是我连文本编辑器也不会用，照着教程下载脚本文件。默默地在terminal上录入bash >curl，告诉我要apt install curl. 这时，我便学会了 apt 相关的命令，发现原来安装软件也是容易的（当然也要网络通畅）。

    ​ 将v2ray按照教程都配置好了后，也确实在运行，可我依然无法科学。（忙活几小时）继续google，找到了v2ray的GUI。Linux配置v2ray详细教程 才知道这个 Qv2ray 按照流程来，也确实看到在运行了，可老是出错，类似于 Failed to read response from … 可我百思不得其解，该做的设置也做了，可怎么就不行呢。直到我查询类似的错误信息时，发现了这个。最终看到了电脑的时间，将时间调准后，成功。 最后顺着这条思路寻找，原来：Ubuntu和Windows默认的时间管理方式不同，所以切换系统发生时间错乱。

    ​ 自此，科学上网的问题大部分被解决。

  2. 文本编辑器。

    ​ 这是一个过不去的坎，我以为Ubuntu的文本编辑跟txt差不多（想多了），当我用vi的时候，我竟一个字都打不出来，基本模式知道，可我用方向键时却出现AB之类的乱码，现在我找到了原因。ubuntu预安装的是tiny版本，所以会导致我们在使用上的产生上述的不便 所以要安装完整的vim。

    ​ 但在这之前，我使用的是 nano 。总算能写一点东西修改点文件了。看了 Nano简单使用 才知道原来的操作也是如此原始（甚至不会整行修改，一直按删除键）。也偶尔用Ubuntu自带的文本编辑器编辑文字，可以更直观的看到文字处理。

    ​ 还有呢，在terminal里面的复制粘贴是 Ctrl+shift+c/v。Ctrl+c是终止操作的执行。可以用上下浏览代码历史，还算是比较方便了。

  3. 输入法。

    ​ 先直奔已经成功的说，最终安装了 fcitx-googlepinyin 。在Ubuntu18.04系统中安装谷歌拼音输入法（Google Pinyin）

    ​ 一开始希望用上搜狗拼音，但出现了奇怪的问题，各种组件崩坏，顺便系统崩了一次无法开机，只能在busybox里面敲代码了（哭）。入坑文：ubuntu 18.04 LTS 安装搜狗输入法

    ​ 可能是完全卸载了ibus框架的原因导致输入法出错，装不上搜狗还只能打出繁体字，最终在我一次尝试后系统无法开机。

    ​ 从头再来。

  4. 更新源。

    ​ 这居然是较晚才知道的步骤，相见恨晚，应该第一时间把Ubuntu的更新源换为国内的，可以少很多麻烦。

    ​ ubuntu18.04 更新源：这一篇提供了很多源。

    ​ Ubuntu18.04下更改apt源为阿里云源：这一篇更细致并且有点高端，作为一个啥都不懂的萌新来说一开始不是很明白他在干嘛，但他提供了另一篇有用的文章：ubuntu sudo update与upgrade的作用及区别

  5. pip源。

    ​ 知道了源后，自然就会想到pip换源。以前不知道pip是什么，但看python相关的文章总有它的身影。Python 之 pip安装 及 使用详解

    ​ 安装OpenCV怎么都装不上呀，网络问题还是存在的。想到了切换国内源，一查也是可行的。Ubuntu16.04下配置pip国内镜像源加速安装

最终总算配置完全。

The-4th-StarStudio-Weekly-Newspaper

发表于 2019-11-23

这周可是咸鱼的一周呢

显然，我这周什么都没干（该死的数学），以至于写不出周报来了。只能乱扯点话萌混过关吧QAQ。

《设计心理学》依然在看第一章

事实上，可能是翻译的问题，书阐释不清，确实对我没有太大的吸引力，仅能从破碎的句子中划一点关键词，简单的明白“示能” 和“意符” 是个不同的东西，也是设计师要重点注意的东西。

但是有看不少绘画论

因为选修课的要求，要写关于绘画的论文，便查找了不少资料（而且还在寻找中），这大概也能提高我的设计素养吧，毕竟设计还是要从艺术品中来，光看目前市面上的主流的所谓“平面设计”，最终也只能流于形式，为了设计而设计便没有了设计的灵魂，是不具有生命力的。

以及做了一点练习

帮某人P图，来满足体育课的要求，换取一杯奶茶。

任务要求：30+张keep的3km跑步截图。

任务分析：

  • 截图背景是固定的，就有了可操作空间。
  • 截图需要不同的数字，数字字体显然是无法找到的，就从截图中挨个抠出来。
  • 需要30+张的不同的截图

任务难度：大量单一操作，以及满足手机截图的高分辨率，同时具有一定的无序性。

项目实施：

  1. 向甲方索要带有所有元素的截图（数字、标点、背景色）。

  2. 以一张背景为底板，涂掉所有可变与需变元素。

  3. 抠出所有的数字与符号。根据大小形式单独成组。

  4. 建立相对无序的数据组（配速、公里、时长、卡路里）

  5. 以公里数分类，各制作10+份。（其中可以以相同小数值不同公里数批量导出）

  6. 打包成压缩包。（.7z）

制作耗时：2h。

流程构思，项目分析耗时：2d。

最后的不足

依然不会用Ai和Ps做图标，这可怎么办呢咕咕咕。

本周任务：一个LOGO

前期准备：

  • 在网上寻找相关的logo寻找灵感，关键词：star, stars, stars logo.

  • 选取各logo中主要的元素进行重构

  • 产生最终构思（形状，色调，字体）

中期制作：

构思完logo的大致样式了后，制作的步骤就会有针对性。因此先查找有关的ps、ai教程，熟练操作。

但是！

发现并不能足够高效的用ps和ai。(:з」∠)

在查找相关教程后，宣布放弃。

改用 Autodesk SketchBook , 直接画出来。

因此，在一通操作后：

  1. 画出logo的背景形状，填充颜色。
  2. 画出logo内部的主体内容（星星），填充颜色（此处我错误的滥用莫兰迪色，产生土俗效果）
  3. 加上名字（StarStudio）
  4. 添加光晕

成品（说实话改进空间巨大，以后再重制），耗时 2h。

后期反思：

虽然在deadline之前成功出稿子，但成品仍显得很不成熟，模仿痕迹还是很重，色彩的选取不够和谐。

硬伤是并没有用上ps和ai，即所学的知识并未派上实际用场，未能转化为生产力。

不过对笔的使用倒是更熟练了，大部分线条都是一遍过，不用多次修改或是重画。手更稳了（笑）。

然后萌发了想买各种带手写笔的电子产品的想法。完了以后要吃土了（逃）。

其他

  • 换了新的梯子，稳定快速并且贵(:з」∠)

Tutorial Video： How to EASILY Draw 2D landscapes in Photoshop

预期效果

其中使用到多种有用且好记的快捷键，以及一种选色技巧。

  • Window> Find extension exchange> Adobe color theme。
    • 其中有选色rear。
    • 只需选择一个颜色即可创造不同灰度与色调的颜色。
  • 一切行动前先新建图层（Ctrl + right click）
  • 使用套索工具（L）勾出轮廓，”Then Fill a layer”(Alt + backspace)[foreground] or (Ctrl + backspace)[background] 并选择solid color。
  • 在 Color picker 中只选中S 或 B 调整 亮度与更深/浅的颜色。
  • 调出curve（Ctrl + M）
  • “Do some shading here”遮罩图层：新建图层与上> Alt + click。
  • 制作树笔刷：Filter> Render> Tree（然后添加到笔刷就完了）可调整扰动和大小。
